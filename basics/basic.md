## Variables in Swift
1. Integer
2. Double
3. Float
4. Boolean
5. String

## Creating variables 
```swift
var num = 5
var num : Int
```
## Naming Convention
In swift, we use camel case convention for writing the variables and functions
```swift
var pressedButton
var randomNumber
```

## Constants
For a varaible, whose value will remain constant throughout the code we use let keyword
```swift
let age = 19
```
